CSS Grid

O que é CSS Grid?



O CSS Grid é um sistema de layout bidimensional que possibilita a criação
de estruturas de páginas organizadas em linhas e colunas de maneira simples e
flexível. Ele simplifica o posicionamento de elementos na tela, possibilitando a
criação de layouts que vão desde os mais simples até os mais complexos, com
maior controle do que outros métodos, como float ou flexbox.


Funções principais



Grid-template-columns

Basicamente define o número e tamanho das colunas da grade, possui algumas variações, como por exemplo...

grid-template-columns: auto / Cria colunas com tamanho proporcional ao contêiner e à tela.

Grid-template-rows

É uma propriedade CSS usada para definir quantas linhas o layout Grid terá e a altura de cada uma delas.

grid-template-rows: auto / Cria linhas verticais com tamanho proporcional ao contêiner e à tela.

Grid-template-areas



é uma propriedade do CSS Grid Layout que oferece uma maneira,
intuitiva e visual de definir a estrutura do seu layout de grade
nomeando as áreas dentro do grid. Em vez de posicionar os itens
individualmente com linhas e colunas, você pode desenhar o layout
diretamente no seu CSS, usando nomes para cada área.
